Batch Version (leave on Any if unsure)

纯原 Pure OG — best quality. Original factory molds + materials. Same factory as retail. PK/OG/LJR batches.
公司级 Company — 95%+ accuracy. Head-layer leather. Hard to tell apart from retail. Best value.
真标 True Label — real tags, decent quality. Good daily wear. Can spot differences on close inspection.
通货 Common — basic quality. Avoid for passing — obvious differences on close look.

⚠️ Real 纯原 cannot pass authentication apps — any supplier claiming 100% pass rate on Goat/StockX is likely lying.
